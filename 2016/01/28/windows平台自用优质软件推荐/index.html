2016-01-28 • 标签: windows 优质软件

自用优质软件

由于有多年装机经验
各种软件使用测试的也比较多
现把个人平时离不开的优质软件整理一下

压缩解压

windows系统自带有zip格式的归档，win8之后更有虚拟光驱集成，那么平时需要解压也就剩rar格式和7z了，7z作为开源软件，损坏文件的修复一直是个头疼的事，而winrar不支持修改7z。完美的，好像找不到。好压则是国产带头之作，个人推荐。国外还有款小众的，更推荐大家使用。

2345好压

Bandizip

图片浏览

windows自带有照片查看器，不支持gif，不能顺滑缩放，切换图片太麻烦，
总之很难用。关注小众软件的大概知道很多人用谷歌收购的picasa，但是这货不支持gif。。又是格式问题。这也是注重版权的结果。国内的，2345的看图王，虽然这货名字难听，但是算法是很好的，缩放自如，最近又加上了pdf查看的功能，而且自动保存阅读进度哦。同样，有款小众的更清爽。

2345看图王

Honeyview

至于其他的，见仁见智吧。

win10其实就用系统的够了，词库自然就差点了，要完美适配的，目前据我所知，只有已经归搜狗家的QQ输入法纯净版。win7，我还是建议用百度输入法吧。
